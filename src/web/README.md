# 网页客户端（Phaser 3）

## 环境要求

- [Node.js](https://nodejs.org/) 18+（安装后终端可用 `npm -v`）

## 命令

```powershell
cd E:\CCfirst\src\web
npm install
npm run dev
```

## 猫咪格图

将 PNG 放到仓库根目录：

- `assets/cats/board/tile_own.png`
- `assets/cats/board/tile_enemy.png`
- `assets/cats/board/tile_neutral.png`

刷新页面即可；未放置时使用程序生成的色块六边形。
