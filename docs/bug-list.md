# 《咪城大师》问题清单

当前暂无新确认 bug。

## 已修复

1. Laya3D 连续建造后可能报 `Cannot set properties of null (setting 'localPosition')`，原因是翻转动画完成时整层重建销毁了动画节点；已改为动画结束不整层重建，并在节点销毁时清理计时器。
2. Laya3D 图纸金币数字在运行视角下可能把 `50/25/250` 读成 `20/52/520`；已改为固定读数方向，并在价格前增加金币图标。
3. Laya3D 玩家在红方地面建造实体建筑后地块不翻绿；已改为建造落地时立即把建筑所在格改为建筑所属阵营。
4. Laya3D 顶部 HUD 大面板遮挡红方行动区；已改为顶部轻量状态栏、半透明资源区和底部功能区布局。
5. Laya3D 士兵攻击敌方建筑未毁就占领建筑格；已改为先破对应贴边墙、再打建筑本体，建筑毁后才翻色。

1. Laya3D 图纸建造待 LayaAir 点击建造验收与细化；当前应点击绿方主塔周边带贴地图纸图案、金币图标和金币额度的六边形地板触发翻转建造。
2. Laya3D HUD/经济待 LayaAir 运行验收与细化；当前已按 720×1280 有效游戏画面布局，显示金币、倒计时、占格数和提示，并支持建造扣金币、基础产金、金矿加产金。
3. Laya3D 战斗占地闭环待 LayaAir 运行验收与细化；当前含贴边墙、破墙、建筑毁后占地、红方最小 AI。
4. Laya3D 胜负判定。
5. Laya3D 贴边墙美术与建筑毁坏过程动画细化。
