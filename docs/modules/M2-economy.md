# M2：局内经济 + 开格 + 问号盲盒

## 玩法（1:1）

- 局内金币随时间/矿山增加（开局给初始金）。
- 点击相邻可开格：扣费 → 固定池出建筑/空地，或问号池盲盒（见 `docs/random-rules.md`）。
- 建筑：防御塔、兵营、矿山（矿山进入 CD 产金）。
