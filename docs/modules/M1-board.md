# M1：棋盘与占格

## 玩法（与占城大师 1:1）

- 六边形地图；每格有归属：中立 / 玩家 A / 玩家 B。
- 单位进入敌方格 → 该格归属变为该单位所属玩家（M3 实现移动，M1 只做数据与显示）。

## 程序交付

- [ ] 六边形坐标与像素转换
- [ ] 渲染棋盘（占位色或你的格图）
- [ ] 点击格高亮（为 M2 开格做准备）
- [ ] 占格 API：`setTileOwner(q,r, playerId)`
