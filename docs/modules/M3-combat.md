# M3：自动战斗与爆兵

## 玩法（1:1）

- 兵营：按 CD 在邻格或指定点生成单位。
- 单位：自动寻路敌建筑/最近敌人，普攻，进入敌方格占格。
- 防御塔：射程内自动攻击地面/空中（若卡表区分）。

## 【M3 再索要】美术

每个兵种一个文件夹，例如 `assets/cats/units/cat_sword/`：

- `idle.png` 或 `walk_01~04.png`（可先 1 张静图）
- 可选：`attack.png`

**MVP 只需 1 个兵种 + 1 种塔** 即可验证推进感。

## 需确认

1. 第一个兵种名称（对应 IP，如「剑士猫」）→ 定 `unitId`：`cat_swordsman`  
2. 移动速度、血量、攻击：先用占城大师量级，你之后只改 JSON。
