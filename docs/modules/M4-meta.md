# M4：局外 — 宝箱 / 卡牌 / 主城 / 战力

## 玩法（1:1 方向）

- 对战结束 → 获得宝箱（槽位上限）。
- 等待 CD → 开箱得卡、金币、木材。
- 卡牌升级 → 提升属性（战力公式汇总）。
- 匹配：按战力区间找对手（MVP 可先人机）。

## 【M4 再索要】美术

| 资产 | 说明 |
|------|------|
| 卡面 × N | 与 `data/cards.json` 的 id 一致 |
| 宝箱 普通~传奇 | 4 档或 1 档通用 |
| 主城 | 局外 hub 背景或建筑 |
| 金币/木材 | 局外货币图标 |

## 网页 MVP 简化

- 无广告加速；开箱仅倒计时。
- 领地扩地（木材开格）→ **M5 二期**，先只做卡牌+主城等级。
