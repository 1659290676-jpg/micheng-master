# 咪城大师 · 建筑实体分类（GDD）

## 1. 核心与经济类

| 实体 | ID | 机制 |
|------|-----|------|
| 领主塔 | `lord_base` | HP 100；失败判定；基础产金 +5/秒 |
| 金矿 | `gold_mine` | 图纸 50 金固定；全局 +2 金/秒；低血量 |

## 2. 静态防御类

| 实体 | ID | 机制 |
|------|-----|------|
| 基础箭塔 | `arrow_tower` | 图纸 50 金；快攻单体 |
| 范围炮塔 | `aoe_tower` | 进阶；慢攻 AOE（预留） |

## 3. 盲盒兵营类

| 品质 | 建筑 | 产出单位 |
|------|------|----------|
| 普通 | `barracks_melee` | 持剑步兵 `melee` |
| 稀有 | `barracks_ranged` / `barracks_flying` | 弓箭手 / 飞龙 `flying` |
| 史诗 | `barracks_melee` | 重装 `heavy` |
| 传奇 | `barracks_ranged` | 弩车 `siege`（拆塔加成） |

抽卡档位：25 / 50 / 100 / 250 金（见 `data/blueprint_weights.json`）

## 4. 城墙（自动生成）

| 实体 | ID | 机制 |
|------|-----|------|
| 城墙 | `wall` | 非图纸；建筑群外缘自动铺设；阻挡地面近战 |

## UI 与占城大师对齐

- 图纸显示在**格子平面内**（非悬浮）
- 格子尺寸加大（`HEX_SIZE = 38`）
- 占位使用**建筑实体贴图**，非纯色圆块
